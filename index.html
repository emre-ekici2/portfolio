Foodbuddy

Description

An individual project in which I was tasked with creating a WPF application of my own choosing. I chose to create a food and weight logger. The application allows the user to track their daily caloric intake and their weigh-ins. Then the application will take the inputs into account and over time provide an accurate caloric intake for the user to lose weight at a healthy pace.

Technologies used

  • C#
  • WPF
  • SQLite
  • Github actions

Personal contribution

I accomplished with the following:

  • Creating the user interface for inputs and displays
  • Writing the code behind which tracks/stores all data and makes calculations for caloric values
  • Placing the application in a Github actions pipeline to automatically update the application on Github
